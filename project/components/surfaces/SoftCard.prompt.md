A flat, tinted surface with no shadow or border — the quiet counterpart to `Card`. Use for secondary, grouped content (info rows, summaries, inset blocks).

```jsx
<SoftCard>
  <span className="cu-eyebrow">Plano</span>
  <div>Bradesco Saúde · 047.812.099-3</div>
</SoftCard>
```

Defaults to the warm linho tint (`--c-surface`); pass `background` to use a palette accent-soft tone.
